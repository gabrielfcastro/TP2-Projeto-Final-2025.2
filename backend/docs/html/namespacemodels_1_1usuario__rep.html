models.usuario_rep.adicionar_usuario ( str  email,
str  senha,
str  nome,
str  tipo_usuario 
)

Adiciona os dados do usuário na tabela do banco de dados.

Parameters
emailCorresponde ao email do usuário.
senhaCorresponde a senha do usuário
nomeCorresponde ao nome do usuário
tipo_usuarioCorresponde ao papel do usuário
Precondition
email Deve conter padrao email : parte-local@dominio, parte-local pode conter letras, numeros, hifen (-) e ponto (.), dominio pode conter letras, numeros e hifen (-) separados por ponto (.), deve conter no máximo 64 caracteres.
senha Deve conter pelo menos 1 letra Maiúscula, 1 letra Minúscula, 1 numérico e 1 caractere especial e deve conter entre 8 à 15 caracteres.
nome Deve conter apenas letras e espaço, sem acentuação e deve conter entre 2 à 20 caracteres.
tipo_usuario Deve ser um dos seguintes: Cliente, Feirante, Admin.
Postcondition
Cria um novo registro na tabela 'usuarios' com os dados fornecidos.
O campo 'senha' no banco armazena o hash bcrypt, e não a string original.
Exceptions
ValueErrorSe o formato dos padrões das variáveis tiverem inválidos.
Returns
Retorna um dicionário contendo os dados públicos do usuário criado (cpf, email, nome).
