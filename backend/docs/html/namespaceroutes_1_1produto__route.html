routes.produto_route.criar_produto ( )

Cria um novo produto no sistema.

Esta rota processa uma requisição POST proveniente do front contendo os dados necessários para criar um novo produto.

Parameters
[in]feirante_idID do feirante que está cadastrando o produto.
[in]nomeNome do produto.
[in]precoPreço do produto.
[in]descricaoDescrição do produto (opcional).
[in]latitudeLatitude da localização do produto (opcional).
[in]longitudeLongitude da localização do produto (opcional).
Precondition
feirante_id Deve ser um ID válido de um feirante existente.
nome Deve conter entre 5 e 50 caracteres.
preco Deve ser um valor maior que zero e conter exatamente duas casas decimais.
descricao Deve conter no máximo 500 caracteres, se fornecida.
Postcondition
Insere um novo produto na tabela 'produtos' do banco de dados.
Exceptions
ValueErrorSe os dados fornecidos forem inválidos.
ConnectionErrorSe houver falha ao salvar o produto no banco.
Returns
Retorna um JSON de resposta contendo os dados do produto criado em caso de sucesso (201) ou uma mensagem de erro em caso de falha.
